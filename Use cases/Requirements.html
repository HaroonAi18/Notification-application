Project Requirements

Project Overview

This project involves the development of a desktop application for sending user notifications in case of forgetting to call a contact in a while.

Functional Requirements

  • The application must monitor incoming calls and identify missed calls that is integrated to various  connecting applications like WhatApp, Messenger, Instagra, X and Telegra .
  • Users should be able to set notification preferences, such as time intervals for notifications.
  • Notifications should be user-configurable, including message content and notification frequency.
  • Notifications should be sent through a user-defined channel (e.g., pop-up notifications, email).
  • Users must have the option to snooze or dismiss notifications.

Non-Functional Requirements

  • The application must be user-friendly and have an intuitive user interface.
  • It should run efficiently and consume minimal system resources.
  • The application should be compatible with major desktop operating systems (e.g., Windows, macOS, Linux).
  • Data related to calls and notifications should be securely stored and protected.
  • The application should provide error handling and logging for troubleshooting.
